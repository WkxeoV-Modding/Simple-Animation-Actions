# Simple-Animation-Actions
A script to make animation action set up super simple!

Again this should automatically set up an action named Animation Menu to whatever your main active page is.

Basically, you create your animations in blockbench. You can make as many as you want.
The name of the animations have to follow a simple rule.
groupname_animationname
The names cannot have spaces or underscores (I will clarify this script is in my back burner).

The script than finds groupname and creates an action based off that name.
It then takes whatever animations start with that group name and creates a set of actions under that action.

It is very crude and half assed. Do not expect this to update in a while if not at all.
